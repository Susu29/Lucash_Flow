LucashFlow — Accounting software project

Small accounting software built with Django (Python) & PostgreSQL, covering transactions, ledger, balance, and financial statements. Deployed on AWS EC2.

Usage of AI

AI has been used like a tutor, basically like an enhanced google. It's usage has been limited.
By querying google, the AI Overview greatly helped me saving time browsing multiple websites.
Basically giving me more exemples that Django documentation often lacks of.

Cases where I relied on ChatGPT:

  • Logo:
    The logo was made using ChatGPT.

  • Sample database:
    The sample database was made using ChatGPT, as creating it manually would have been very tedious.
    Following the prompts, I have made a script verifying the integrity of the csv file given.
    Script that has proven to be useful as I had to modify the prompt numerous time.

  • CSS / layout:
    Since CSS beyond the basics was not a priority, I’ve asked a lot of help on how to adjust the tables as they subjects to a lot of misalignment.

  • Django InlineFormSet:
    Some guidance on the InLineFormSet as I didn’t found the documentation very explicit.

FAQ

Why is the website in english but the accounts in french ?
I am more familiar with French accounting standards (PCG), so I chose to apply the French accounting structure to this project.
However, the website is in English to reach a larger public.

Source of the PCG : (Accounting Plan)
https://github.com/Seb35/plans-comptables-francais

French translation :

  • Accounts : Comptes
  • Suppliers : Fournisseurs
  • Customers : Clients
  • Transactions : Ecritures comptables
  • Ledger : Grand livre
  • Balance : Balance comptable
  • Income statement : Compte de résultat
  • Balance sheet : Bilan
